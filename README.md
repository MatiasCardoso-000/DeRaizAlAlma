# De Raiz al Alma | Constelaciones y Tarot

Un espacio moderno, elegante y espiritual para el autoconocimiento y la sanación sistémica.

## Secciones Principales

- **Carta Astral**: Mapa energético profundo para descubrir dones y propósito.
- **Constelaciones Familiares**: Trabajo sistémico para sanar el linaje y patrones heredados.
- **Revolución Solar**: Guía astrológica anual para alinear tu energía con el nuevo ciclo.
- **Tarot Evolutivo**: Herramienta de autoconocimiento centrada en el presente y el empoderamiento personal.

## Características Técnicas

- **Framework**: React + Vite
- **Estilos**: Tailwind CSS
- **Navegación**: React Router (navegación integral sincronizada)
- **Tipografía**: Cinzel (Serif) y Montserrat (Sans-serif)
- **Componentización**: Header global reactivo con estado dinámico de enlaces.

## Instalación

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`
